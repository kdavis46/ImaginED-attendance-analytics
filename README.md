# ImaginED-attendance-analytics
Business Intelligence dashboard analyzing attendance trends, engagement metrics, and program effectiveness using Power BI, SQL, and KPI reporting.
---

## Project Overview

This project was designed to evaluate relationships between attendance, engagement, and program participation across multiple educational support programs.

The dashboard provides visibility into:
- attendance trends
- student engagement levels
- school participation metrics
- absence rates
- program effectiveness
- engagement vs attendance relationships

---

## Tools & Technologies

- Power BI
- SQL
- DAX
- Excel

---

## Dashboard Features

### Program Effectiveness Analysis
Analyzed relationships between engagement scores and attendance rates across educational programs.

### Attendance Distribution
Compared attendance levels across participating schools and educational sites.

### Engagement Analysis
Evaluated student engagement performance across multiple enrichment and intervention programs.

### Attendance Status Reporting
Tracked present vs absent participation metrics across schools.

---

## Key Findings

- Attendance rate measured approximately 51%
- Absence rate measured approximately 49%
- Average engagement score measured 2.92
- Engagement and attendance relationships varied significantly between programs

---

## Dataset Disclaimer

This project uses a synthetic dataset generated for portfolio and demonstration purposes.

The data was structured to simulate attendance and engagement reporting scenarios commonly used in nonprofit and educational program analytics.
